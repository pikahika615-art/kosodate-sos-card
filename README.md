# 子育てSOSカード

## 概要
子育て中に「今すぐ助けてほしいこと」を  
ワンタップで相手に伝えられるSOSカードアプリです。  
スマホでの利用を想定し、操作が直感的で見やすいUIを意識して制作しました。

## 制作背景・目的
子育て中は、言葉で状況を説明する余裕がない場面が多くあります。  
そのような場面でも、必要なサポート内容をすぐに伝えられる仕組みがあれば便利だと感じ、
シンプルな操作で使えるアプリとして本作品を制作しました。

## 主な機能
- SOS内容をボタンで選択
- 選択した内容をメッセージとして表示
- クリア操作でサンキューカードを表示
- レスポンシブ対応（スマートフォン利用を想定）
- PWA対応（ホーム画面に追加可能）

## 使用技術
- HTML
- CSS  
  - レスポンシブデザイン  
  - アニメーション表現
- JavaScript  
  - DOM操作  
  - イベント処理
- Git / GitHub
- GitHub Pages
- PWA（manifest.json）

## 工夫した点・学んだこと
- スマートフォンでの操作性を優先し、文字サイズやボタン配置を調整しました
- 画面遷移を最小限にし、迷わず使えるUIを意識しました
- GitHubを使ったバージョン管理を行い、競合解消やマージ作業も経験しました
- 実際にスマホで動作確認を行いながら改善を重ねました

## 今後の改善予定
- SOS内容のカスタマイズ機能
- デザインやアニメーションの強化
- 使用シーンに応じた表示パターンの追加
